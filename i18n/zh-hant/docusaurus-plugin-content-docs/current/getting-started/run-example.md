# 運行範例

1. 克隆 wuba/react-native-echarts 存儲庫

```shell
git clone git@github.com:wuba/react-native-echarts.git
```

2. 安裝依賴項

```shell
yarn
```

3. 轉到示例目錄

```shell
cd example
```

4. 執行 ios 或 android

```shell
yarn ios
# or
yarn android
```

就是這樣！ 您應該看到以下螢幕：

| iOS | Android |
| --- | --- |
| ![ios](./ios.png) | ![android](./android.jpg) |

恭喜你！ 您已成功執行示例。
